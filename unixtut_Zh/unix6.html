UNIX 教程第六节

其它有用的 UNIX 命令

quota

所有学生都在文件系统上为他们的个人文件分配了一定数量的磁盘空间，通常大约100Mb。 如果您超过配额，则会有7天时间删除多余的文件。

要检查您当前的配额以及您使用了多少配额，请键入

% quota -v

df

df 命令报告文件系统上剩余的空间。 例如，要了解文件服务器上剩余的空间，请输入

% df .

du

du 命令输出每个子目录使用的千字节数。 如果您已超过配额，并且想要找出哪个目录包含的文件最大，则非常有用。 在您的主目录中，输入

% du

compress

此命令减小了文件的大小，从而释放了宝贵的磁盘空间。 例如，输入

% ls -l science.txt

并记下文件的大小。 然后压缩 science.txt，输入

% compress science.txt

这将压缩文件并将其放在名为 science.txt.Z 的文件中。

要查看大小的变化，请再次输入 ls -l。

要解压缩文件，请使用 uncompress 命令。

% uncompress science.txt.Z

gzip

此命令也压缩文件，并且比 compress 更有效。 例如，要压缩 science.txt，请键入

% gzip science.txt

这将压缩文件并将其放在名为 science.txt.gz 的文件中

要解压缩文件，请使用 gunzip 命令。

% gunzip science.txt.gz

file

file 根据它们包含的数据类型对命名文件进行分类，例如 ascii（文本），图片，压缩数据等。要报告主目录中的所有文件，请键入

% file *

history

C shell 保留了您输入的所有命令的有序列表。 每个命令根据输入的顺序给出一个数字。

% history (show command history list)

如果您使用的是C shell，则可以使用感叹号（！）轻松再调用命令。

% !! （回调最后一个命令）

% !-3 （回调起最近的第三个命令）

% !5 （回调历史记录的第三个命令）

% !grep （回调最近的以 grep 起始的命令）

您可以改变历史记录缓存的大小，输入

% set history=100
