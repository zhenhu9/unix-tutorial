排版约定

在下文中，我们将使用以下排版约定：

  • 用粗体打印字体书写的字符是可以在计算机中输入的命令。
  • 用斜体打印字体书写的字符表示非特定文件或目录名称。
  • 在方括号中插入的单词[Ctrl]表示要按下的键。

例如：

% ls anydirectory [Enter]

表示 "在 UNIX 提示符 %，输入 ls 紧随目录名，然后输入标记为 Enter 的键。"

不要忘了按 [Enter] 键：在完成此操作之前，命令不会发送到计算机。

注意：UNIX 是区别大小写的，因此 LS 与 ls 不同。

这同样适用于文件名，因此 myfile.txt，MyFile.txt 和 MYFILE.TXT 是三个单独的文件。请注意，如果复制文件到另一台电脑，DOS 和 Windows 无法区分大小写。
