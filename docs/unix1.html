UNIX 教程一

1.1 列出文件和目录

ls （list）

首次登录时，您当前的工作目录是您的主目录。 您的主目录与您的用户名同名，例如：ee91ab，它是您保存个人文件和子目录的位置。

要找出主目录中的内容，请键入

% ls （list 缩写）

ls 命令列出当前工作目录的内容。

您的主目录中可能没有可见的文件，在这种情况下，将返回 UNIX 提示符。 另外，在创建帐户时，系统管理员可能已经添加了一些文件。

事实上，ls 不会导致主目录中的所有文件被列出，而只会列出那些名称不以点（.）开头的文件。 以点（.）开头的文件称为隐藏文件，通常包含重要的程序配置信息。 它们是隐藏的，因为你不应该更改它们，除非你非常熟悉 UNIX！！！

要列出主目录中的所有文件，包括名称以点开头的文件，请键入

% ls -a

ls 是命令可以使用选项的示例：-a 是选项的例子。 选项会改变命令的行为。 有一些在线手册页可以告诉您特定命令可以采用哪些选项，以及每个选项如何改变命令的行为。 （参见本教程后面的内容）

1.2 创建目录

mkdir （make directory）

现在，我们将在主目录中创建一个子目录，以保存您将在本教程中创建和使用的文件。 要在当前工作目录中创建一个名为 unixstuff 的子目录，请键入

% mkdir unixstuff

要查看刚刚创建的目录，请键入

% ls

1.3 切换到其他目录 

cd （change directory）

命令 cd directory 意味着将当前工作目录切换到‘directory’。 当前工作目录可以被认为是您所在的目录，即您在文件系统树中的当前位置。

要切换到刚刚创建的目录，请键入

% cd unixstuff

输入 ls 以查看内容（应该为空）

1.4 目录 . 和 ..

仍在 unixstuff 目录中，键入

% ls -a

如您所见，在 unixstuff 目录（以及所有其他目录中）中，有两个名为（.）和（..）的特殊目录。

在UNIX中，（.）表示当前目录，因此键入

% cd .

注意：cd 和 . 之间有一个空格。

意味着留在原地（unixstuff目录）。

这一开始可能看起来不太有用，但使用（.）作为当前目录的名称将节省大量的输入，我们将在本教程后面看到。

 

（..）表示当前目录的父级，因此键入

% cd ..

将带您到层次结构中上一层的一个目录（返回到您的主目录）。 现在就试试。

注意：键入不带参数的 cd 总是会返回到您的主目录。 如果您在文件系统中迷失，这非常有用。

1.5 路径名

pwd（print working directory）

路径名可以让你搞清楚你在整个文件系统相关的位置。 例如，要找出主目录的绝对路径名，请键入 cd 以返回到主目录，然后键入

% pwd

完整路径名看起来像这样 -

/a/fservb/fservb/fservb22/eebeng99/ee91ab

这意味着 ee91ab（您的主目录）位于目录 eebeng99（组目录）中，该目录位于 fservb 文件服务器上。

注意：

/a/fservb/fservb/fservb22/eebeng99/ee91ab

可以缩短为

/user/eebeng99/ee91ab

1.6 有关主目录和路径名的更多信息

了解路径名

首先输入 cd 以返回主目录，然后键入

% ls unixstuff

列出 unixstuff 目录的内容。

 

现在键入

% ls backups

你会收到这样的消息 -

backups: No such file or directory

原因是，backups 不在您当前的工作目录中。 要对不在当前工作目录（您当前所在的目录）中的文件（或目录）上使用命令，您必须 cd 到正确的目录，或指定其完整路径名。 要列出 backups 目录的内容，必须键入

% ls unixstuff/backups

 

~ （你的主目录）

主目录也可以由波浪号 ~ 字符引用。 它可用于指定从主目录开始的路径。 因此键入

% ls ~/unixstuff

将列出 unixstuff 目录的内容，无论您当前在文件系统中的哪个位置。

你认为会显示什么

% ls ~

将列出?

你认为会显示什么

% ls ~/..

将列出?

总结

ls 显示文件和目录
ls -a 显示所有的文件和目录
mkdir 创建目录
cd directory 切换到指定的目录
cd 切换到主目录
cd ~ 切换到主目录
cd .. 切换到父目录
pwd 显示当前目录的路径
